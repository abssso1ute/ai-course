# Lighthouse — скриншоты «до» и «после»

## Как получить отчёт

1. Соберите production и запустите превью:

```bash
npm run build
npm run preview
```

2. Откройте `http://127.0.0.1:5173` в Chrome.
3. DevTools → **Lighthouse** → режим *Navigation* → категории *Performance*, *Accessibility* (опционально Best practices / SEO).
4. Сохраните скриншот итоговой страницы отчёта: **до оптимизаций** (если есть ветка/коммит без `v-memo` и без `manualChunks`) и **после** (текущий `main`).

## Куда положить файлы

Сохраните изображения в эту папку:

- `lighthouse/before.png`
- `lighthouse/after.png`

Так вы выполните формальный артефакт задания для блока 3.

## Подсказки по улучшению метрик

- Уменьшайте начальный JS: ленивые маршруты, контроль зависимостей.
- Следите за **INP**: не выполняйте тяжёлую работу в каждом `input` без `debounce`.
- Фиксируйте высоту области чата, чтобы снизить **CLS** при появлении новых сообщений.
