# UI Kit Specification — SmartService

## 1. Button
- Mobile: full-width, крупная зона нажатия
- Tablet: средний размер, возможны иконки + текст
- Desktop: компактные кнопки, hover и secondary варианты

## 2. Card
- Mobile: одна колонка, вертикальная структура
- Tablet: 2 колонки
- Desktop: 3–4 колонки, hover эффекты

## 3. Navigation
- Mobile: hamburger menu
- Tablet: компактное меню с иконками
- Desktop: горизонтальное меню

## 4. Input Form
- Mobile: вертикальное расположение, full width
- Tablet: 2 колонки полей
- Desktop: структурированные группы полей

## 5. Grid System
- Mobile: 1 колонка
- Tablet: 2–3 колонки
- Desktop: 4–12 колонок

## 6. Typography
- Mobile: уменьшенные размеры
- Tablet: стандартная иерархия
- Desktop: крупные заголовки, усиленная иерархия
