# UI Kit (module_2.3)

## Структура компонентов

### Атомы
Базовые элементы интерфейса.
Примеры:
- Button.vue
- Input (если добавлен)

Использование:
<Button variant="primary">Кнопка</Button>

---

### Молекулы
Комбинации атомов с логикой.
Пример:
- LoginForm.vue

Использование:
<LoginForm />

---

### Организмы
Полноценные UI-блоки.
Пример:
- Modal.vue

Использование:
<Modal :isOpen="true">
  <LoginForm />
</Modal>

---

## Принципы работы с ИИ

1. Чем точнее промпт — тем лучше результат с первого раза
2. Код от ИИ всегда требует ревью и доработки
3. Генерация — это старт, а не финальная версия продукта
